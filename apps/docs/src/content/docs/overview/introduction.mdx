## What You Can Build

Lightfast Chat serves as both a ready-to-use application and a foundation for building:

- **Customer Support Systems**: Integrate AI assistants into your support workflow
- **Educational Platforms**: Create interactive learning experiences with AI tutors
- **Research Tools**: Build specialized interfaces for AI-assisted research
- **Creative Applications**: Develop writing assistants, code helpers, and more

## Technology Stack

Built on a foundation of modern, production-ready technologies:

- **Frontend**: Next.js 15 with App Router and React 19
- **Backend**: Convex for real-time database and serverless functions
- **Styling**: Tailwind CSS v4 with shadcn/ui components
- **Language**: TypeScript with strict mode
- **Build**: Turborepo with Bun for fast, cached builds
- **Quality**: Biome for linting and formatting
