# Chat Features

Lightfast Chat offers a rich set of features designed to enhance your AI conversation experience.

## Thread-Based Conversations

### Creating New Threads

- Click the "New Chat" button to start a fresh conversation
- Each thread maintains its own context and history
- Threads are automatically saved and can be resumed later

### Thread Management

- **Rename threads**: Click on the thread title to edit
- **Delete threads**: Use the delete option in thread settings
- **Search threads**: Find conversations by content or title

## AI Model Selection

### Supported Models

- **Claude Sonnet 4**: Anthropic's most capable model
- **GPT-4**: OpenAI's flagship model
- **GPT-3.5 Turbo**: Fast and cost-effective option

### Switching Models

1. Click the model selector in the chat interface
2. Choose your preferred AI model
3. The selection applies to new messages in the current thread

### Model-Specific Features

Each model has unique strengths:

- **Claude Sonnet 4**: Excellent for reasoning, analysis, and coding
- **GPT-4**: Great for creative writing and general conversations
- **GPT-3.5 Turbo**: Quick responses for simple queries

## Message Features

### Rich Text Support

- **Markdown rendering**: Format your messages with markdown
- **Code highlighting**: Syntax highlighting for code blocks
- **Mathematical expressions**: LaTeX math rendering
- **Links and images**: Clickable links and image previews

### Message Actions

- **Copy message**: Copy AI responses to clipboard
- **Regenerate response**: Get a new response for the same prompt
- **Edit and resend**: Modify your message and get a new response

## Resumable Streaming

### Real-time Responses

- Messages stream in real-time as the AI generates them
- See responses appear word-by-word for better engagement
- Streaming continues even if connection is temporarily lost

### Connection Recovery

- Automatic reconnection on network issues
- Messages resume from where they left off
- No data loss during disconnections

## Feedback System

### Rating Responses

Rate AI responses with thumbs up/down:

1. Hover over any AI message
2. Click the thumbs up (üëç) or thumbs down (üëé) icon
3. Optionally provide detailed feedback

### Detailed Feedback

When rating a response, you can provide:

- **What worked well**: Highlight positive aspects
- **What needs improvement**: Suggest specific improvements
- **Alternative suggestions**: Provide better response examples

## Advanced Features

### Message History

- Full conversation history maintained
- Search within conversations
- Export conversations (coming soon)

### Keyboard Shortcuts

- **Enter**: Send message
- **Shift + Enter**: New line in message
- **Ctrl/Cmd + /**: Open command palette
- **Ctrl/Cmd + N**: New thread

### Context Management

- Each thread maintains separate context
- Context is preserved across sessions
- Smart context truncation for long conversations

## Tips for Better Conversations

### Effective Prompting

1. **Be specific**: Provide clear, detailed instructions
2. **Use examples**: Include examples of desired output
3. **Set context**: Explain the purpose and audience
4. **Iterate**: Refine prompts based on responses

### Model Selection Tips

- Use **Claude Sonnet 4** for complex reasoning and coding tasks
- Choose **GPT-4** for creative and conversational content
- Select **GPT-3.5 Turbo** for quick, simple queries

### Managing Long Conversations

- Start new threads for different topics
- Summarize key points periodically
- Use clear headings and structure in your messages

## Troubleshooting

### Common Issues

**Slow responses**: Check your internet connection and try a different model.

**Message not sending**: Refresh the page and try again.

**Context lost**: Long conversations may hit context limits; start a new thread.

### Getting Help

- Use the feedback system to report issues
- Check the network indicator for connection status
- Contact support through the help menu
