# Welcome to Lightfast Chat

Lightfast Chat is a modern, feature-rich AI chat application built with the latest web technologies. It provides a seamless experience for interacting with multiple AI models from different providers.

## Key Features

- **Multi-Provider AI Support**: Chat with Claude Sonnet 4, GPT models, and more
- **Resumable Streams**: Never lose your conversation progress
- **Thread-based Conversations**: Organize your chats with a clean thread system
- **Feedback System**: Rate and provide feedback on AI responses
- **Real-time Updates**: Powered by Convex for instant synchronization
- **Modern UI**: Built with Next.js 15, Tailwind CSS, and shadcn/ui

## Tech Stack

- **Frontend**: Next.js 15 with App Router and Partial Prerendering (PPR)
- **Backend**: Convex for real-time database and API
- **Styling**: Tailwind CSS v4.x with shadcn/ui components
- **AI Integration**: AI SDK with multi-provider support
- **TypeScript**: Strict mode with comprehensive type safety
- **Package Manager**: pnpm for fast, efficient dependency management
